# todoor

Single page, javascript todo app that saves it's todo's in local storage. Uses:

* require js
* bootstrap
* moment js
* underscore js
* store js
* bootstrapValidator
* mousetrap
* jquery
* jquery ui

# Getting it to work

1. Go to public folder and get the bower dependencies with:

   ```bash
   bower install
   ```
   
2. Go to application root director and execute:

   ```bash
   sh serve.sh
   ```
